## 실습 1 ✍️ 회의록 자동 정리 스킬 만들기

### 📌 상황 설명

수아의 팀은 채용 면접관 조율 회의, 교육 기획 회의, 복지 제도 논의 회의 등  
매주 다양한 회의를 진행한다. 회의록 양식은 팀마다 다르고,  
KT HR팀은 **참석자 / 핵심 안건 / 결정사항 / 담당자별 액션 아이템** 형식을 사용한다.  
이 양식에 맞춰 자동으로 정리해주는 스킬을 직접 만들어보기로 했다.

### 🔧 1단계 — 스킬 크리에이터로 스킬 만들기

새 채팅을 열고 아래 프롬프트를 참고해 클로드에게 스킬 제작을 지시합니다.  
회의록 양식은 여러분 회사 양식에 맞게 전달하세요.

> ℹ️ **[사용자 지정(Customize) → 스킬]의 [예시] 탭에서 `skill-creator`가 활성화된 상태여야 합니다.**

### 💬 프롬프트 예시

```
스킬 크리에이터 스킬을 사용해서 'KT HR팀 회의록 자동 정리 스킬'을 만들어줘.

우리 팀 회의록 양식은 아래와 같아:
- 회의명 / 일시 / 장소
- 참석자 (이름, 소속, 역할)
- 핵심 안건 (번호순 정리)
- 결정사항 (안건별 결론)
- 담당자별 액션 아이템 (담당자 / 내용 / 완료 기한)
- 다음 회의 예정일

입력된 회의 녹취록에서 위 양식에 맞춰 내용을 자동으로 추출하는 지침을 넣어줘.
결과물은 다운로드할 수 있게 하나의 압축 파일로 줘.
```

### ✅ 스킬 크리에이터가 만들어주는 것

클로드가 아래 내용을 자동으로 코딩합니다.

- `SKILL.md` — 스킬의 동작 방식과 지침을 담은 핵심 파일
- 회의록 양식 추출 로직
- 워드(.docx) 파일로 출력하는 기능
- 압축 파일(`.zip`) 형태로 패키징

## 실습 2 📦 완성된 커스텀 스킬 설치하고 결과물 확인하기

### 🔧 2단계 — 스킬 설치하기

클로드가 코딩을 마치면 **오른쪽 아티팩트 패널에서 다운로드 버튼**을 눌러 압축 파일을 저장합니다.

1. **[사용자 지정(Customize) → 스킬]** 메뉴로 이동합니다.
2. **[+ → 스킬 업로드]**를 클릭합니다.
3. 다운로드한 `.zip` 파일을 **드래그 앤 드롭**합니다.
4. 스킬 목록에 `KT-HR-회의록-자동정리`가 표시되면 설치 완료입니다.

### 🔧 3단계 — 결과물 확인 및 다운로드

새 채팅을 열고 회의 녹취록을 붙여넣어 스킬을 사용해봅니다.

### 💬 프롬프트 예시

```
KT-HR-회의록-자동정리 스킬을 사용해서 아래 회의 녹취록을 정리해줘.

[회의 녹취록]
일시: 2026년 6월 24일 오후 2시
참석: 이수아(HR기획팀 주임), 김태호(채용팀장), 박수연(교육기획 과장)

이수아: 오늘은 하반기 신입 공채 일정 확정하고, 면접관 교육 계획도 같이 논의해요.
김태호: 서류 접수는 8월 1일부터 2주간으로 하면 좋겠어요. 작년보다 1주 늘리고요.
박수연: 면접관 교육은 1차 면접 전주 수요일 오전에 2시간으로 잡겠습니다.
이수아: 지원자 안내 공고문은 제가 7월 25일까지 초안 작성할게요.
김태호: 외부 채용 플랫폼 등록은 박 과장님이 맡아주시겠어요?
박수연: 네, 7월 28일까지 원티드, 사람인, 링크드인 동시 등록할게요.
이수아: 다음 회의는 7월 10일 같은 시간에 할까요?
```

## 실습 3 🗂️ 코워크 시작하기 — 파일 시스템 개념 이해

### 📌 상황 설명

수아는 매달 생성되는 채용 서류, 교육 자료, 회의록 파일이 바탕화면과 문서 폴더에  
뒤죽박죽 쌓여간다. 코워크를 사용하면 클로드가 수아의 컴퓨터 폴더에 직접 접근해  
파일을 읽고, 이동하고, 수정하는 작업을 말 한마디로 처리해준다.

### 📌 코워크가 뭔가요?

**코워크(Cowork)** 는 클로드가 여러분의 컴퓨터에 있는 파일과 폴더에 직접 접근하는 기능입니다.

지금까지는 클로드에게 작업을 시키기 위해 파일을 하나하나 업로드해야 했는데,  
이제는 "이번 달 채용 서류 폴더 정리해줘"라고 말만 하면  
클로드가 직접 폴더를 열어보고 작업할 수 있습니다.

| 구분 | 일반 채팅 | 코워크 |
|------|-----------|--------|
| 파일 처리 | 파일을 직접 업로드해야 함 | 폴더를 지정하면 직접 접근 |
| 작업 결과 | 방법만 알려줌 | 실제로 파일을 읽고, 이동하고, 수정 |
| 활용 예시 | "회의록 정리 방법 알려줘" | "HR 폴더 회의록 파일 전부 양식에 맞게 정리해줘" |

### 📂 파일 시스템 기본 개념

수아의 컴퓨터 폴더 구조를 예시로 살펴보겠습니다.

```
💻 드라이브 (서류 캐비닛 전체)
│
├── 📁 바탕화면
│   ├── 채용공고_초안_v3.docx
│   └── 면접평가표_최종.xlsx
│
├── 📁 문서
│   └── 📁 HR기획팀
│       ├── 📁 2026_하반기_공채
│       │   ├── 회의록_0624.docx
│       │   └── 채용일정표.xlsx
│       └── 📁 교육기획
│           └── 면접관교육_커리큘럼.pptx
│
└── 📁 다운로드
```

### 📌 핵심 개념 정리

| 개념 | 설명 | 수아의 예시 |
|------|------|------------|
| **드라이브(Drive)** | 서류 캐비닛 자체. 윈도우는 `C:\`, macOS는 `/` | `C:\` |
| **폴더(Folder)** | 캐비닛 안의 서랍. 폴더 안에 폴더를 만들 수 있음. 디렉터리라고도 함 | `HR기획팀` |
| **파일(File)** | 서랍 안의 실제 서류. 문서, 사진, 음악 등 모든 것 | `회의록_0624.docx` |
| **경로(Path)** | 파일을 찾아가는 주소 | `C:\Users\이수아\문서\HR기획팀\회의록_0624.docx` |

### 📌 자주 쓰는 파일 확장자

| 확장자 | 파일 종류 | 수아의 업무 파일 예시 |
|--------|-----------|----------------------|
| `.docx` | 워드 문서 | 회의록, 채용공고문, 합격 통보 메일 |
| `.xlsx` | 엑셀 | 채용 일정표, 지원자 현황, 예산 집행 내역 |
| `.pptx` | 파워포인트 | 면접관 교육 자료, 경영진 보고 자료 |
| `.pdf` | PDF | 입사지원서, 계약서, 공식 문서 |

## 실습 4 📁 작업 폴더 선택하고 권한 부여하기

### 📌 상황 설명

수아는 코워크를 본격적으로 사용하기 위해 채용 관련 파일이 모여 있는  
`HR기획팀` 폴더를 작업 폴더로 지정하려 한다.

### 🔧 코워크 작업 폴더 설정 방법

**01단계** 클로드 데스크톱 앱을 실행하고 왼쪽 패널에서 **[코워크(Cowork)]**를 클릭합니다.  
입력창에서 **[프로젝트에서 작업 → 폴더 선택]**을 누릅니다.

> ⚠️ 윈도우 환경이라면 코워크가 바로 실행되지 않을 수 있습니다.  
> '가상 머신 플랫폼을 사용할 수 없음' 창이 나타나면 **[활성화]**를 누르고 권한을 허용한 후  
> 클로드 데스크톱 앱을 다시 실행하면 정상적으로 실행됩니다.

**02단계** 시스템 파일 선택 대화 상자가 나타나면 작업할 폴더를 선택합니다.  
수아는 실습을 위해 `HR_Cowork`라는 새 폴더를 만들고 선택합니다.

> ⚠️ 홈 디렉터리처럼 너무 많은 파일이 들어 있는 경로는 선택하지 마세요.  
> 코워크가 파일을 분석하는 시간이 아주 오래 걸립니다.

**03단계** 권한 요청 창이 나타나면 **[항상 허용]** 또는 **[허용]**을 누릅니다.

```
Claude가 "HR_Cowork"의 파일을 변경하도록 허용하시겠습니까?

여기에는 모든 파일과 하위 폴더가 포함됩니다.
Claude는 파일을 읽고, 편집하고, 영구적으로 삭제할 수 있으며,
연결된 서드파티 도구와 파일 콘텐츠를 공유할 수 있습니다.
민감한 정보 노출에 주의하세요.

[취소]  [항상 허용]  [허용]
```

**04단계** 폴더 선택이 완료됐다면 간단하게 테스트해봅니다.


### 📌 코워크 사용 시 주의사항

코워크는 복잡한 작업을 수행할 때 진행 상황과 작성 중인 파일을 **오른쪽 패널에서 실시간**으로 보여줍니다.

> ⚠️ 클로드가 건드리지 않아도 되는 파일을 건드리거나,  
> 진행 상황에 보이는 작업 단계가 생각한 것과 다르다면 **과감하게 작업을 중지**하세요.  
> 클로드의 실수를 빠르게 잡아내야 시간과 토큰을 아낄 수 있습니다.

|------|------------------|
| ✍️ 스킬 크리에이터 | KT HR팀 회의록 양식 맞춤 스킬 직접 제작 |
| 📦 커스텀 스킬 설치 | zip 파일 드래그 앤 드롭으로 간편 설치 |
| 📝 회의록 자동 정리 | 녹취록 붙여넣기 → 워드 파일 자동 생성 |
| 🗂️ 코워크 파일 시스템 | 드라이브·폴더·파일·경로 개념 이해 |
| 📁 작업 폴더 권한 설정 | HR_Cowork 폴더 지정 후 클로드에 접근 권한 부여 |

> ✅ 코워크 모드는 클로드가 로컬 파일 시스템에 직접 접근하는 기능입니다.  
> ✅ 작업 폴더를 선택해 권한을 부여하면 클로드가 파일을 읽고 쓸 수 있습니다.  
> ✅ 탐색기나 파인더에서 마우스로 직접 하던 작업을 말로 쉽게 자동화할 수 있습니다.  
> ✅ 작업 폴더를 최소 범위로 지정하고, 민감한 파일은 명시적으로 제외해 보호할 수 있습니다.
