## 실습 1 🗄️ 파일 시스템 연결 — 채용 서류 폴더 정리하기

### 📌 상황 설명

수아의 바탕화면에는 지난 3개월간 채용 시즌을 거치며 쌓인 파일들이 난장판이다.  
`최종이력서_홍길동_v3.pdf`, `면접평가표_수정본_진짜최종.xlsx`, `채용공고_초안_백업2.docx`...  
어디서부터 손대야 할지 모를 정도다.

### 🔧 커넥터 연결 방법

1. 클로드 데스크톱 앱 → 하단 프로필 → **[설정]** 클릭
2. **[커넥터 → 커넥터 둘러보기]** 선택
3. 검색창에 `Desktop Commander` 입력 후 선택
4. **[설치]** 버튼 클릭 → **사용 중** 상태 확인

> ⚠️ 데스크톱 커맨더 커넥터는 **클로드 데스크톱 앱에서만** 사용 가능합니다.

### 💬 프롬프트 예시

```
데스크톱 커맨더 커넥터를 사용해서 바탕화면을 정리해줘.
채용 관련 파일, 교육 기획 파일, 기타 파일로 분류해줘.
```

### ✅ 예상 결과

클로드가 바탕화면을 스캔한 후 아래와 같이 폴더를 자동 생성하고 파일을 분류합니다.

```
📁 채용_서류/
   └── 이력서, 면접평가표, 채용공고 파일
📁 교육_기획/
   └── 교육 커리큘럼, 강사 섭외 자료
📁 기타_파일/
   └── 분류 불가 파일
```

### 💡 수아의 한마디
> "3개월 치 채용 파일을 30초 만에 정리해줬어요. 진짜 감동..."

## 실습 2 📂 구글 드라이브 연결 — 채용 기획안 찾고 요약하기

### 📌 상황 설명

수아는 이번 달 KT 하반기 공채 준비를 위해 지난해 채용 기획안을 찾아야 한다.  
파일 제목이 정확히 기억나지 않아서 구글 드라이브를 한참 뒤지고 있던 참이었다.

### 🔧 커넥터 연결 방법

1. **[설정 → 커넥터]** 메뉴 진입
2. **Google Drive** 항목의 **[연결]** 클릭
3. 구글 계정 선택 후 로그인
4. **[모두 선택]** 눌러 권한 허용 → **[데스크톱 앱 열기]** 클릭
5. 새 채팅 입력창 **[+ → 커넥터 → Google Drive]** 토글 활성화 확인

### 💬 프롬프트 예시

```
내 구글 드라이브에서 작년에 작성한 KT 하반기 채용 관련 기획안을 찾아줘.
제목에 '공채' 또는 '하반기'가 들어간 것 같아.
```

### ✅ 예상 결과

```
구글 드라이브를 검색했습니다. 다음 문서들을 찾았어요.

1. 2024 KT 하반기 공채 기획안_v2.docx
   - 수정일: 2024년 8월 20일 / 크기: 3.1MB

2. KT 하반기 채용 일정표.xlsx
   - 수정일: 2024년 8월 14일 / 크기: 512KB

3. 하반기 면접관 교육자료.pptx
   - 수정일: 2024년 9월 3일 / 크기: 8.7MB

어떤 문서의 내용을 확인해드릴까요?
```

### 💬 이어서 요약 요청

```
1번 문서를 요약해줘. 주요 채용 일정과 전형 단계를 중심으로.
```

### 💡 수아의 한마디
> "드라이브 검색창에서 한참 찾던 걸 클로드한테 말 한마디 했더니 바로 찾아주고 요약까지 해줬어요."

## 실습 3 📧 지메일 연결 — 채용 메일 분류하고 답장 쓰기

### 📌 상황 설명

오늘 수아의 메일함에는 지원자 문의 메일, 헤드헌터 연락, 내부 협업 요청, 각종 뉴스레터가 뒤섞여 47통이 쌓여 있다.  
일일이 열어보고 답장을 쓰다 보면 오전이 다 간다.

### 🔧 커넥터 연결 방법

1. 왼쪽 패널 **[사용자 지정(Customize)]** 메뉴 클릭
2. **[커넥터 → Gmail → 연결]** 클릭
3. 구글 계정 선택 후 **[모두 선택 → 계속]** 눌러 권한 승인
4. 클로드 데스크톱 앱으로 복귀 → Gmail 토글 활성화 확인

> ℹ️ [모두 선택]을 누르지 않으면 기능이 제한될 수 있습니다.

### 💬 프롬프트 예시 1 — 메일 현황 파악

```
받은 메일 중 읽지 않은 메일을 모두 확인해줘.
발신자, 제목, 중요도를 표로 정리해줘.
```

### ✅ 예상 결과

| # | 날짜 | 발신자 | 제목 | 중요도 |
|---|------|--------|------|--------|
| 1 | 6/24 | 박지훈 (지원자) | KT 하반기 서류 접수 확인 문의 | 🔴 높음 |
| 2 | 6/24 | 이재원 팀장 | 면접관 교육 일정 협의 요청 | 🔴 높음 |
| 3 | 6/23 | 헤드헌터 A사 | IT 직군 경력직 추천 건 | 🟡 중간 |
| 4 | 6/23 | HR 뉴스레터 | 2025 채용 트렌드 리포트 | 🟢 낮음 |

### 💬 프롬프트 예시 2 — 메일 자동 분류

```
읽지 않은 메일을 채용 문의, 내부 협업, 헤드헌터, 뉴스레터/광고로 분류해줘.
카테고리별로 몇 통인지도 알려줘.
```

### 💬 프롬프트 예시 3 — 답장 초안 작성

```
박지훈 지원자 메일에 답장을 써줘.
서류는 정상 접수되었고, 결과는 2주 내 개별 안내 예정이라고 정중하게 전해줘.
```

### ✅ 예상 답장 초안

```
제목: Re: KT 하반기 서류 접수 확인 문의

박지훈 님,

안녕하세요. KT 인재경영본부 HR기획팀 이수아입니다.

지원해 주셔서 감사합니다.
서류는 정상적으로 접수되었으며, 검토 후 2주 내에 개별적으로 결과를 안내드릴 예정입니다.

궁금한 점이 있으시면 언제든지 문의해 주세요.

감사합니다.
이수아 드림
```

> ⚠️ 클로드가 작성한 메일은 반드시 직접 검토 후 발송하세요. AI는 언제든 실수할 수 있습니다.

### 💡 수아의 한마디
> "매일 아침 30분씩 쓰던 메일 확인 시간이 5분으로 줄었어요. 답장 초안도 뽑아줘서 수정만 하면 돼요."

## 실습 4 📅 구글 캘린더 연결 — 면접 일정 등록하기

### 📌 상황 설명

수아는 이번 주 1차 실무 면접 일정을 잡아야 한다.  
면접관 3명의 일정을 모두 고려해야 하는데, 캘린더를 일일이 확인하다 보면 시간이 훌쩍 지나버린다.

### 🔧 커넥터 연결 방법

1. **[사용자 지정 → 커넥터 → Google Calendar → 연결]** 클릭
2. **[모두 선택]** 눌러 권한 승인
3. 새 채팅 입력창 **[+ → 커넥터 → Google Calendar]** 토글 활성화 확인

### 💬 프롬프트 예시 1 — 현재 일정 확인

```
구글 캘린더에서 이번 주 일정을 확인해줘.
```

### ✅ 예상 결과

```
이번 주(6/23~6/27) 일정은 총 5개입니다.

오늘 (6/24 화)
- 채용공고 초안 검토 — 오전 10:00~11:00
- 인사팀 주간 회의 — 오후 2:00~3:00 / 본사 15층 회의실

수요일 (6/25)
- 면접관 사전 교육 — 오전 10:00~12:00

목요일 (6/26)
- 경영진 보고 자료 준비 — 오후 1:00~4:00

금요일 (6/27)
- 상반기 채용 결과 정리 — 종일
```

### 💬 프롬프트 예시 2 — 빈 시간 확인 및 일정 등록

```
다음 주 화요일 오후 2시에 1차 실무 면접을 2시간으로 잡고 싶은데,
기존 일정과 겹치는지 확인하고, 괜찮으면 구글 밋 링크도 포함해서 일정 등록해줘.
면접관은 김태호 팀장, 박수연 과장, 최민준 대리야.
```

### ✅ 예상 결과

```
화요일 오후 2시는 기존 일정과 겹치지 않습니다. 일정을 생성할게요.

📅 KT 하반기 공채 1차 실무 면접
- 일시: 7/1(화) 오후 2:00~4:00
- 참석자: 김태호 팀장, 박수연 과장, 최민준 대리
- Google Meet 링크 포함

참석자 초대 메일도 함께 보낼까요?
```

### 💬 프롬프트 예시 3 — 일정 삭제 (일정 취소 시)

```
지원자 측 사정으로 취소됐어. 다음 주 화요일 1차 실무 면접 일정 삭제해줘.
```

> ℹ️ 삭제 전 클로드가 재확인을 요청합니다. 되돌릴 수 없으므로 신중하게 승인하세요.

### 🔒 권한 관리 팁

불필요한 삭제가 걱정된다면 권한을 미리 해제해두세요.

**[사용자 지정 → 커넥터 → Google Calendar → Delete Calendar Event → ⊘ 비활성화]**
