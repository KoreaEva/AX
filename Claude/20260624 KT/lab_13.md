## 실습 1 🔧 클로드 포 크롬 설치하기

### 📌 상황 설명

수아는 채용 공고 작성, 후보자 리서치, 교육 자료 수집 등 웹 브라우저를 하루 종일 사용한다.  
매번 클로드 탭을 따로 열어 내용을 복사해서 붙여넣는 게 번거로웠는데,  
크롬 확장 프로그램으로 설치하면 어느 페이지에서나 바로 클로드를 쓸 수 있다는 걸 알게 됐다.

### 🔧 설치 방법

**01단계** 크롬 웹스토어에서 `Claude` 확장 프로그램을 검색합니다.

**02단계** [Chrome에 추가] 버튼을 누릅니다.
- 확인창에서 [확장 프로그램 추가]를 누릅니다.
- 이어서 나오는 화면에서 [승인]을 누르면 설치 완료입니다.

> ⚠️ 처음 사용할 때는 신뢰할 수 있는 웹사이트에서만 권한을 부여하세요.  
> 금융이나 개인정보, 업무 중요 작업을 처리하기 전에는 반드시 클로드의 동작을 확인하세요.

**03단계** 크롬 브라우저 오른쪽 위의 [퍼즐 아이콘(🧩)]을 눌러 확장 프로그램 목록에서 [Claude]를 확인합니다.

**04단계** [Claude]를 클릭하면 오른쪽에 클로드 창이 나타납니다.  
안내 문구 확인 후 [시작하기]를 누르면 바로 사용할 수 있습니다.

### 💡 수아의 한마디
> "크롬에 설치하고 나니 어느 페이지에서나 클로드를 바로 쓸 수 있어요. 탭 왔다 갔다 안 해도 돼서 너무 편해요!"

## 실습 2 📱 HR 트렌드 SNS 타임라인 요약하기

### 📌 상황 설명

수아는 HR 트렌드를 파악하기 위해 LinkedIn, Threads 등 SNS를 매일 확인한다.  
하지만 피드를 끝도 없이 스크롤하다 보면 시간이 훌쩍 지나버린다.  
클로드 포 크롬을 이용하면 타임라인을 자동으로 훑고 핵심만 요약해준다.

### 💬 프롬프트 예시

```
linkedin.com 열어서 오늘 HR/채용 관련 타임라인 훑어보고 요약해줘.
```

### 📋 진행 순서

1. 클로드가 **계획(플랜)**을 먼저 세웁니다.
2. 계획에 이상이 없으면 **[플랜 승인]**을 누릅니다.
3. 중간에 권한 승인 요청 화면이 나오면 **[이 작업 허용]** 또는 **[이 사이트에서 항상 작업 허용]**을 선택합니다.

> ℹ️ 브라우저 가장자리가 **주황색**으로 변하면 클로드가 브라우저를 조작 중인 거예요.

### ✅ 예상 결과

클로드가 자동으로 SNS에 접속하고 스크롤을 내리며 화면을 캡처합니다.  
이후 캡처한 이미지를 바탕으로 타임라인 내용을 요약해줍니다.

```
오늘 HR 타임라인 주요 내용 요약입니다.

📌 채용 트렌드
- AI 기반 이력서 스크리닝 도구 도입 기업 증가 (채용 시간 40% 단축 사례 공유)
- MZ세대 이직률 관련 설문 결과: 성장 기회 부재가 1위 이유

📌 조직 문화
- 심리적 안전감이 팀 성과에 미치는 영향 연구 공유
- 재택/출근 혼합 정책 관련 기업별 사례 다수

📌 HR 테크
- 온보딩 자동화 솔루션 신제품 출시 소식
- 성과 관리 시스템에 AI 피드백 기능 추가 트렌드
```

### 💡 수아의 한마디
> "매일 아침 30분씩 SNS 돌아다니던 걸 클로드가 5분 만에 요약해줘요. 이제 진짜 중요한 업무에 집중할 수 있어요."

## 실습 3 🔍 채용 관련 최신 게시글 검색 & 요약하기

### 📌 상황 설명

수아는 이번 KT 하반기 공채 공고를 올리기 전, 경쟁사 채용 트렌드와 직무 설명 키워드를 리서치해야 한다.  
클로드 포 크롬은 단순히 페이지를 보기만 하는 게 아니라, **직접 클릭하고 검색**할 수도 있다.

### 💬 프롬프트 예시

```
LinkedIn에서 '통신사 HR 채용 공고' 최신 게시글을 검색해서 주요 직무 키워드를 요약해줘.
```

또는

```
Threads에서 최신 HR 트렌드 정보 게시글을 검색해서 요약해줘.
```

### ✅ 예상 결과

```
통신사 HR 채용 공고 검색 결과 요약입니다.

자주 등장하는 직무 키워드:
- 채용 기획 및 운영 (JD 작성, ATS 관리)
- HRBP (HR Business Partner)
- 조직문화 / DEI (다양성·형평성·포용성)
- 인재 개발 / L&D (Learning & Development)
- 데이터 기반 HR 분석 (People Analytics)

주요 우대 역량:
- 채용 플랫폼 운영 경험 (원티드, 사람인, LinkedIn 등)
- AI 채용 도구 활용 경험
- HR 데이터 분석 경험 (Excel, Python 기초)
```

### 💡 수아의 한마디
> "경쟁사 채용 공고 일일이 들어가서 복사할 필요 없이, 클로드가 알아서 검색하고 키워드까지 뽑아줬어요."

## 실습 4 ⏰ 반복 리서치 작업 자동화 설정하기

### 📌 상황 설명

수아는 매일 아침 HR 트렌드를 확인하는 루틴이 있다.  
이 작업을 매일 반복하는 대신, 클로드 포 크롬에서 **정해진 시간에 자동으로 실행**되도록 설정해둘 수 있다.

### 🔧 자동화 설정 방법

**01단계** 자동화할 작업을 프롬프트로 입력합니다.

```
매일 아침 9시마다 LinkedIn에서 HR 트렌드 요약해서 알려줘.
```

**02단계** 입력창 오른쪽 위의 **[ ⋮ ]** 버튼을 클릭합니다.

**03단계** **[작업으로 변환]**을 선택합니다.

**04단계** 단축키 만들기 화면에서 설정합니다.

| 설정 항목 | 입력값 |
|-----------|--------|
| 이름 | `/hr-trend` |
| 시작 위치 | `https://www.linkedin.com` |
| 일정 | 매일 / 오전 9:00 |
| 모델 | Claude Sonnet 4.6 |

**05단계** **[단축키 만들기]**를 누르면 설정 완료입니다.

> ℹ️ 토글 버튼이 활성화되어 있어야 일정에 맞춰 자동 실행됩니다.  
> 채팅창에서 `/hr-trend`를 입력하면 언제든지 바로 실행할 수도 있습니다.

### ✅ 자동화 활용 예시

| 단축키 이름 | 실행 내용 | 일정 |
|-------------|-----------|------|
| `/hr-trend` | LinkedIn HR 트렌드 요약 | 매일 오전 9:00 |
| `/job-check` | 채용 플랫폼 신규 지원자 현황 확인 | 매일 오전 10:00 |
| `/edu-news` | 기업 교육/L&D 관련 최신 뉴스 수집 | 매주 월요일 오전 9:00 |

### 💡 수아의 한마디
> "출근하면 이미 HR 트렌드 요약본이 준비돼 있어요. 아침 루틴이 완전히 바뀌었어요."

## 실습 5 🔌 커스텀 커넥터 연결하기

### 📌 커스텀 커넥터란?

구글 워크스페이스, 노션, 슬랙 같은 커넥터는 클릭 몇 번으로 연결할 수 있습니다.  
하지만 세상에는 이보다 훨씬 더 다양한 도구와 서비스가 있습니다.

**커스텀 커넥터**는 개발자 커뮤니티에서 만든 MCP 서버를 직접 설치해 클로드와 연결하는 방식입니다.  
조테로(Zotero), 깃허브(GitHub), 스포티파이(Spotify) 등 다양한 서비스용 커넥터를 찾아 쓸 수 있습니다.

> 💡 커스텀 MCP는 **스미더리(smithery.ai)**에서 검색할 수 있습니다.

### 🔍 안전한 MCP 선택 기준

커스텀 커넥터를 고를 때 아래 항목을 확인하세요.

| 항목 | 확인 내용 |
|------|-----------|
| 기능 | 내가 원하는 작업을 지원하는가? |
| 문서 | 설치 방법과 사용법이 잘 설명되어 있는가? |
| 유지보수 | 최근에 업데이트되었는가? 이슈 응답이 있는가? |
| 인기도 | 별(⭐)과 포크(🍴) 개수가 많은가? |
| 라이선스 | 상업적 사용이 가능한 라이선스인가? |

---

### 📌 실습 예시 — HR 업무에 유용한 커스텀 커넥터 활용

> 아래는 원본의 `daiso-mcp`를 수아의 HR 업무 상황에 맞게 재구성한 예시입니다.  
> 실제 커넥터는 스미더리(smithery.ai) 또는 깃허브에서 검색하여 찾아보세요.

#### 🔧 커스텀 커넥터 연결 방법

**01단계** [사용자 지정(Customize) → 커넥터]로 이동하고 **[+ → 커스텀 커넥터 추가]**를 누릅니다.

**02단계** 이름을 자유롭게 짓고, URL 입력창에 MCP 서버 주소를 붙여 넣은 후 **[추가]**를 누릅니다.

```
예시 URL: https://mcp.aka.page
```

> ℹ️ 별도의 프로그램 설치나 API 키가 필요 없는 커넥터도 있습니다.

**03단계** 새 채팅 입력창에서 **[+ → 커넥터]**를 눌러 추가한 커넥터의 토글이 활성화되어 있는지 확인합니다.

---

#### 💬 HR 업무 활용 프롬프트 예시

**채용 공고 사이트 매장(지역) 검색:**
```
서울 광화문 근처 KT 면접 장소 주변 편의점 재고 확인해줘.
면접관들이 간식 사러 갈 수 있게.
```

**복지 혜택 관련 상품 검색:**
```
다이소에서 파는 사무용품 수납 제품 종류 알려줘.
팀원 책상 정리 이벤트 준비 중이야.
```

**올리브영 재고 확인 (사내 이벤트 선물 준비):**
```
광화문 근처 올리브영에서 핸드크림 선물 세트 재고 있는 곳 알려줘.
HR팀 직원 감사 선물로 쓸 거야.
```

### ✅ 예상 결과 예시

```
광화문 인근 올리브영 5개 매장 재고 확인 결과입니다.

상품명: 이니스프리 그린티 핸드크림 세트 (18,000원)

매장명          재고    픽업
올리브영 광화문점   8개    가능
올리브영 종로점    3개    가능
올리브영 청계점    품절    불가
```

> ℹ️ 커넥터는 한 번 연결해두면 **스마트폰의 클로드 앱**에서도 같은 작업을 수행할 수 있습니다.

### 💡 수아의 한마디
> "팀 간식이나 행사 선물 살 때 매장마다 전화해서 재고 확인하던 게 없어졌어요. 클로드한테 물어보면 바로 나오니까요."
