## 실습 1 🔧 엑셀에 클로드 추가 기능 설치하기

### 📌 상황 설명

동현 팀장은 매 분기 비품 구매 내역과 예산 집행 현황을 엑셀로 정리한다.  
수식이 복잡해질수록 틀릴까봐 긴장하고, 후배에게 물어보기도 민망한 나이가 됐다.  
클로드를 엑셀 안에 설치하면 말로 설명하는 것만으로 수식과 서식을 만들어준다는 걸 알게 됐다.

> ⚠️ 엑셀에서 클로드 추가 기능을 사용하려면 **클로드 Pro 이상 요금제**와 **Microsoft 365 구독**이 필요합니다.

### 🔧 설치 방법

**01단계** 엑셀을 실행합니다. 리본 메뉴에서 **[추가 기능]**을 클릭합니다.  
검색창에 `Claude`를 입력한 후 **'Claude by Anthropic for Excel'**을 찾아 **[추가]**를 누릅니다.

**02단계** 추가하면 리본 메뉴에 **[Claude]** 탭이 새로 나타납니다.  
이 탭을 클릭하면 오른쪽에 클로드 패널이 열립니다.  
**[Log in to Claude]**를 눌러 클로드 계정으로 로그인합니다.

**03단계** 이미 클로드에 로그인된 브라우저에서 실행 중이라면 바로 승인 화면이 나타납니다.  
**[승인]**을 눌러 진행합니다.

**04단계** 코드를 복사하라는 화면이 나옵니다.  
**[Copy sign-in code]**를 눌러 코드를 복사한 후, 엑셀로 돌아와 클로드 패널에 붙여 넣습니다.  
이어서 나오는 질문은 원하는 항목을 선택하거나 **[Skip]**을 눌러 넘어갑니다.

> ℹ️ 추가 기능은 한 번만 설치하면 됩니다. 다음에 엑셀을 실행할 때 자동으로 Claude 탭이 나타납니다.  
> 보이지 않으면 **[삽입 → 내 추가 기능]** 메뉴에서 클로드를 다시 활성화하세요.

### 💡 동현 팀장의 한마디
> "설치가 이렇게 간단할 줄 몰랐어요. 탭 하나 생겼을 뿐인데 엑셀이 완전히 다른 도구가 된 느낌이에요."

## 실습 2 📋 총무 업무용 샘플 데이터 생성하기

### 📌 상황 설명

동현 팀장은 이번 분기 비품 구매 내역을 정리해야 한다.  
실습을 위해 가상의 비품 구매 데이터를 먼저 만들어보기로 했다.

> ℹ️ 이런 반복적인 데이터 생성 작업은 **Sonnet 모델**을 사용하면 빠르고 효율적입니다.

### 💬 프롬프트 예시

```
KT 총무팀의 가상 비품 구매 내역 데이터 100건을 만들어줘.
항목은 구매일, 품목명, 카테고리(사무용품/IT기기/시설/복지용품),
구매처, 수량, 단가, 총금액, 사업소명, 담당자, 예산 항목으로 구성해줘.
```

### 📋 진행 순서

1. 클로드가 직접 엑셀 파일을 편집하기 전에 **권한 부여를 요청**합니다.
2. 중요한 파일이 아닌 실습용이라면 **[Always allow]**를 선택합니다.  
   (중요한 파일이라면 **[Allow once]** 선택 권장)
3. 클로드가 셀에 값을 자동으로 채워 넣고 서식까지 정리해줍니다.

> ℹ️ 권한 승인 모드는 언제든 변경할 수 있습니다.  
> 클로드 입력창의 **[⊕ 버튼]**을 눌러 **[Ask before edits]** 또는 **[Accept all edits]**로 전환하세요.

### ✅ 예상 결과 (샘플)

| 구매일 | 품목명 | 카테고리 | 구매처 | 수량 | 단가 | 총금액 | 사업소명 | 담당자 | 예산항목 |
|--------|--------|----------|--------|------|------|--------|----------|--------|----------|
| 2024-01-05 | A4 용지(박스) | 사무용품 | 교보문고 | 10 | 25,000 | 250,000 | 광화문 | 김민수 | 소모품비 |
| 2024-01-08 | 노트북 충전기 | IT기기 | 삼성전자 | 3 | 55,000 | 165,000 | 분당 | 이수진 | 기기유지비 |
| 2024-01-12 | 형광등 교체 | 시설 | 한국전기 | 20 | 8,000 | 160,000 | 수원 | 박태준 | 시설관리비 |

### 💡 동현 팀장의 한마디
> "예전엔 더미 데이터 만드는 데 한 시간씩 썼는데, 클로드가 100건을 서식까지 예쁘게 만들어줬어요. 심지어 열 너비도 맞춰줬더라고요."

## 실습 3 🎨 조건부 서식 자동으로 만들기

### 📌 상황 설명

동현 팀장은 비품 구매 내역에서 예산 초과 항목이나 특정 기간의 구매 내역을  
색상으로 한눈에 구분하고 싶다. 조건부 서식 설정은 늘 헷갈리는 부분이었다.

### 💬 프롬프트 예시 1 — 기간 기반 강조

```
구매일이 2024년 1분기(1~3월)인 행의 배경색을 하늘색으로 바꿔줘.
```

### 💬 프롬프트 예시 2 — 금액 기반 조건부 서식

```
총금액이 100만 원 이상인 구매 건의 행은 글자 색을 빨간색으로 표시하는 조건부 서식을 만들어줘.
```

> ℹ️ 리본 메뉴에서 **[조건부 서식 → 규칙 관리...]**를 눌러 클로드가 실제로 적용했는지 확인할 수 있습니다.

### 💬 프롬프트 예시 3 — 복합 조건부 서식 한 번에 적용

```
색상 스케일, 데이터바, 아이콘 집합, 수식 기반 조건부 서식을 한 번에 만들어서 적용해줘.
(총금액 히트맵 + 카테고리별 색상 + 예산 항목 등급 구분 등)
```

### ✅ 예상 적용 결과

| 조건부 서식 종류 | 적용 기준 | 시각 효과 |
|-----------------|-----------|-----------|
| 색상 스케일 | 총금액 크기 | 낮을수록 흰색 → 높을수록 진한 주황 |
| 데이터바 | 수량 | 셀 안에 막대로 비율 표시 |
| 아이콘 집합 | 단가 수준 | 🔴🟡🟢 3단계 아이콘 |
| 수식 기반 | 카테고리=IT기기 | 행 전체 노란색 배경 |
| 수식 기반 | 예산 항목=기기유지비 | 행 전체 파란색 글씨 |

### 💡 동현 팀장의 한마디
> "조건부 서식은 늘 규칙 관리 창에서 헤매다 포기했는데, 말로 설명하니까 클로드가 다 만들어줬어요. 팀원들한테 자랑했습니다."

## 실습 4 📈 데이터 분석하고 대시보드 만들기

### 📌 상황 설명

동현 팀장은 이번 분기 비품 구매 내역을 경영진에게 보고해야 한다.  
사업소별, 카테고리별 지출 현황과 예산 대비 집행률을 한눈에 볼 수 있는 대시보드가 필요하다.

### 💬 프롬프트 예시 1 — RFM 스타일 구매 패턴 분석

```
새 시트에 사업소별 비품 구매 패턴을 분석해줘.
최근 구매일, 구매 빈도, 총 지출 금액 기준으로 사업소 등급을 나눠줘.
(예: 고지출/정기구매/소량구매/비활성 등)
```

### ✅ 예상 결과

클로드가 새 시트를 생성하고 아래와 같이 분석 결과를 정리합니다.

|----------|-----------|---------|--------|------|------|
| 광화문 | 2024-03-28 | 42 | 18,340,000 | 고지출 | VIP 관리 필요 |
| 수원 | 2024-03-15 | 28 | 7,120,000 | 정기구매 | 안정적 집행 |
| 분당 | 2024-01-10 | 6 | 980,000 | 비활성 | 집행률 점검 필요 |

### 💬 프롬프트 예시 2 — 동적 대시보드 생성

```
새 시트에 총무팀 비품 관리 대시보드를 만들어줘.
사업소와 카테고리를 드롭다운으로 선택할 수 있게 하고,
선택에 따라 구매 건수 / 총 지출 금액 / 평균 단가 / 예산 항목별 비중 KPI가 자동으로 바뀌게 해줘.
```

## 실습 5 📋 클로드가 엑셀에서 할 수 있는 기능 정리

### 📌 총무팀 업무 관점에서 확인해야 할 기능 범위

모든 커넥터가 그렇듯이, 클로드가 엑셀의 모든 기능을 사용할 수 있는 것은 아닙니다.  
총무팀 업무에서 자주 쓰는 기능 위주로 가능/불가능 여부를 확인하세요.

### ✅ 가능한 기능

| 분류 | 세부 기능 |
|------|-----------|
| 데이터 입력/관리 | 셀 읽기/쓰기, 행·열 삽입·삭제·그룹핑, 시트 관리·병합·틀고정·지우기 |
| 수식/함수 | 기본 집계(SUM, AVERAGE, COUNT), 조건부·찾기·배열 함수, 텍스트·날짜·동적 수식 |
| 서식/디자인 | 글꼴·색상·배경·정렬, 숫자 서식·테두리·열 너비·행 높이 |
| 조건부 서식 | 데이터바·색 스케일·아이콘 세트, 수식 기반 행·셀 강조 |
| 차트 | 기본 차트(2D), 3D·누적·고급 차트, 차트 편집 |
| 피벗 테이블 | 생성, 집계 함수 변경 |
| 데이터 도구 | 드롭다운·정렬·필터·검색 |
| 기타 | 인쇄 설정, 작업 로그 |

### ❌ 불가능한 기능

| 분류 | 세부 기능 |
|------|-----------|
| 불가능 | SmartArt / 지도 차트 / 이미지 삽입 |
| 불가능 | VBA 매크로 실행 / 슬라이서 |
| 불가능 | Power Query / Power Pivot / 파일 다운로드 |

> ⚠️ 클로드가 못하는 작업을 지시하느라 시간과 토큰을 낭비하지 않도록 주의하세요.

## 🔧 클로드 포 엑셀 설정 팁

### 세션 로깅 (Session Logging)

클로드가 수행한 모든 작업을 별도의 시트에 기록하는 기능입니다.  
**[Claude 패널 → Settings → Session logging]** 토글을 켜두면  
클로드가 어떤 수정을 했는지 이력을 남길 수 있습니다.

> 💡 중요한 파일에서 작업할 때 반드시 켜두는 것을 권장합니다.

### 파일 간 작업 (Let Claude work across files)

**[Claude 패널 → Settings → Let Claude work across files]** 토글을 켜면  
클로드가 현재 열려 있는 **모든 엑셀 파일을 동시에 참조**합니다.  
심지어 파워포인트까지 연동됩니다.

> 💡 비품 구매 내역 파일과 예산 현황 파일을 동시에 열어두고 작업하면 더욱 효율적입니다.

### 💡 동현 팀장의 한마디
